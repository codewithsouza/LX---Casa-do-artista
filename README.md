# LX — Casa do Artista

Site oficial do projeto **LX**, com home e página dedicada de **Mídia Kit**.

## Como rodar localmente

- Abra o arquivo `index.html` no navegador.
- A página de mídia kit está em `midia-kit.html`.

## Estrutura do projeto (clean)

```
/
├─ index.html
├─ midia-kit.html
├─ assets/
│  └─ img/
│     ├─ hero/       # GIF e imagens do hero
│     ├─ albums/     # capas de álbuns
│     ├─ singles/    # capas de singles
│     └─ ui/         # elementos de interface (logos etc.)
```

## Rotas principais

- `index.html` → home do projeto
- `midia-kit.html` → mídia kit
